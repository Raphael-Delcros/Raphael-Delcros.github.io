Lecteur CD

Pour l'apprentissage de Qt (framework de développement d'applications multiplateformes), nous devions créer une application spécifique en fonction de l'année. Pour la nôtre, c'était la reproduction d'un lecteur CD physique. L'utilisateur devait pouvoir manuellement ouvrir et fermer le lecteur, insérer ou retirer le disque, se déplacer entre les différentes musiques, etc. Chaque bouton analogue était remplacé par son équivalent dans le logiciel.

Tags : C++, Qt, Structure MVP

Space Invaders

Notre but était d'importer une carte de la ville de Bayonne pour tracer le chemin le plus court entre certaines œuvres d'art publiques. Nous avons donc utilisé les algorithmes de Dijkstra, BFK et A* pour y parvenir. Malheureusement, nous n'avons pas pu concrétiser ce projet par manque de temps.

Tags : Python, Algorithmes de tri

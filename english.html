Projects

Passatemps

Passatemps is a three year long project about a second-hand board game reselling website, kind of like Vinted or France's Leboncoin. As it is a long project, it was done in a group. I am considered the P.O. of the project. We've had to do the whole concept from scratch, this includes making a graphic chart, mock-ups and wireframes, as well as a 120 pages long document describing in minute details the project.

Tags : UML, HTML, CSS, Web Design, Project Management, MVC, php, MySQL

Lecteur CD

To learn Qt (a multi-platform application development framework), we had to create a specific application depending on the year. For ours, it was a reproduction of a physical CD player. The user had to manually open and close the disk cover, insert or remove the disk, navigate between different songs, etc. Each analog button was replaced by its equivalent in the software.

Tags : C++, Qt, MVP Structure

Space Invaders

Our goal was to import a map of the city of Bayonne to trace the shortest path between certain public artworks. We therefore used the Dijkstra, BFK and A* algorithms to achieve this. Unfortunately, we were unable to complete this project due to lack of time.

Tags : Python, Sorting algorithms
